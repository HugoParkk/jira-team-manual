사이드바

사이드바는 왼쪽에 위치하며, Jira의 메뉴를 선택할 수 있습니다.

메뉴는 다음과 같습니다.

  • 타임라인
  • 백로그
  • 보드
  • 이슈

  • 프로젝트 페이지
  • 바로 가기 추가
  • 프로젝트 설정

메뉴

Jira 사이드바에 있는 메뉴입니다.

타임라인

타임라인에서는 에픽을 추가하고 관리 할 수 있습니다.

타임라인에는 에픽(Epic)들과 에픽의 하위 이슈들이 나열되어서 보입니다.

또한 에픽과 에픽의 하위 이슈들이 기한이 간트차트에 보입니다.

왼쪽에 보이는 에픽 및 이슈들을 클릭하면 상세 정보를 볼 수 있습니다.

이슈에는 진행 상태와 우선 순위, 레이블, 기한, 설명, 담당자, 보고자, 하위 이슈, Spring 정보가 있습니다.

백로그

백로그에서는 스프린트를 만들고 관리 할 수 있습니다.

백로그에는 스프린트들과 스프린트의 이슈들이 나열되어서 보입니다.

백로그의 이슈를 클릭하면 오른쪽에 이슈의 상세 정보가 나옵니다.

보드

보드에서는 현재 스프린트의 이슈들이 뜨고, 이 이슈들의 진행 상태를 관리 할 수 있습니다.

보드의 이슈를 드래그하여 옮기면 진행 상태가 변경 됩니다.

이슈

이슈에서는 필터를 걸어 이슈를 검색 할 수 있습니다. 이슈를 클릭하면 이슈 상세 정보가 보입니다.

이슈 메뉴는 이슈들을 검색할 때 주로 쓰입니다.

프로젝트 페이지

프로젝트 페이지는 Atlassian의 Confluence로 연결 됩니다.

바로 가기 추가

바로 가기 추가는 웹 페이지 등 자주 사용하는 리소스의 링크 또는 Git 리포지토리를 추가 할 수 있습니다.

프로젝트 설정

프로젝트 설정에서는 현재 있는 프로젝트의 설정을 변경 할 수 있습니다.
